# Hello

## Description
The program defines a simple function named **hello()** that print the message **"Hello"** to the console.  
The purpose of this exercise is to practise creating a function, writing output using `cout`, and calling a function from inside `main()`.


## How to run
1.- Open the terminal in VS Code.  
2.- Compile de program:  

g++ Hello.cpp -o Hello    
3.- Execute:    
- On Windowns  
.\Hello.exe  
- On MacOS/ Linux  
./Hello

## What I learned

- How to write and define and a `function` in C++.
- How to use `cout` to print messages.
- How to call a function from inside the `main()` function.
- How to stucture a very simple C++ program with clean and readable code.
- How to use **`void`** this function does **`NOT` return any value**.
